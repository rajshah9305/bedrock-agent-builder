## 🚀 Quick Start Commands

```bash
# Clone and setup
git clone https://github.com/rajshah9305/bedrock-agent-builder.git
cd bedrock-agent-builder
cp env.example .env

# Install dependencies
pnpm install

# Start development (full stack)
pnpm dev:full

# Build for production
pnpm build

# Deploy to Vercel
vercel --prod
```
